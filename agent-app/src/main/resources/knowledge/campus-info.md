# XX 大学校园信息指南

## 🏫 校园建筑与设施

### 食堂
- **南食堂**：位于南校区操场旁，共3层。1楼为大众餐饮，2楼为风味小吃，3楼为教工餐厅。
- **北食堂**：位于北校区图书馆对面，共2层。1楼为清真餐厅，2楼为自选餐厅。
- **西食堂**：位于西校区实验楼旁，以面食和轻食为主。

### 教学楼
- **第一教学楼**：南校区正门入口左侧
- **第二教学楼**：北校区图书馆东侧
- **第三教学楼（逸夫楼）**：南校区西侧，紧邻南食堂
- **实验楼**：西校区，计算机和物理实验课在此上课

### 图书馆
- **主馆**：北校区中心位置，共6层
  - 1楼：总服务台、自习室（24小时开放）
  - 2-3楼：社科类图书
  - 4-5楼：理工类图书
  - 6楼：电子阅览室、研讨间
- **分馆**：南校区第一教学楼2楼

### 行政办公
- **行政楼**：南校区正门右侧
- **各学院办公室**：
  - 计算机学院：实验楼3楼
  - 外国语学院：第二教学楼5楼
  - 经济管理学院：行政楼4楼
  - 理学院：实验楼6楼

### 体育设施
- **南操场**：含足球场、田径跑道
- **北操场**：含篮球场、排球场
- **体育馆**：北操场旁，含羽毛球场、乒乓球室、游泳馆

### 学生服务
- **学生事务中心**：行政楼1楼
- **校医院**：南校区东门旁
- **心理咨询中心**：学生宿舍7号楼1楼
- **快递驿站**：北校区西门旁

## 📅 常见流程

### 选课
- 每学期第1-2周为选课周
- 公共选修课通过教务系统在线选课
- 体育选修课选课时间：每学期第1周，通过体育部网站选课
- 选课网址：http://jwxt.xxu.edu.cn

### 请假
- 1-3天：辅导员审批
- 3-7天：学院副院长审批
- 7天以上：学院院长审批

### 图书借阅
- 借阅期限：30天
- 可续借1次，续借期15天
- 超期罚款：0.1元/天
- 预约图书：可通过图书馆网站或公众号预约
